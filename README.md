# Kubar Labs Website v2

> India's First MSME-Focused Embedded Credit Hub

## Overview

Redesigned website for [Kubar Labs](https://kubar.tech) featuring a "Precision Finance" dark theme with embedded credit infrastructure messaging.

## Tech Stack

- **Vanilla HTML/CSS/JS** — no framework dependencies
- **Space Grotesk** (display) + **Inter** (body) + **JetBrains Mono** (code)
- **Fluid type scale** with `clamp()` for responsive typography
- **4px spacing system** with CSS custom properties
- **CSS-only scroll reveal** animations
- **Canvas particle network** in hero section

## Design System

| Token | Value |
|-------|-------|
| Primary | `#00D4AA` (Electric Teal) |
| Secondary | `#FFB547` (Warm Amber) |
| Background | `#0A0A0F` (Near Black) |
| Surface | `#12121A` |
| Text | `#F5F5F7` |

## Sections

1. **Hero** — Particle canvas, gradient headline, trust strip
2. **Problem** — $300B credit gap counter animation
3. **Protocol** — 3-step process (Connect, Match, Disburse)
4. **Intelligence Engine** — 4 technology cards
5. **NavDhan** — Flagship product with live dashboard mockup
6. **Solutions** — Tab toggle (Banks/NBFCs vs MSMEs)
7. **Impact** — Animated counter stats
8. **Team** — Core team + Advisory board
9. **Resources** — Podcast, Newsletter, Knowledge Base
10. **Contact** — Form + contact channels

## Development

Serve locally:
```bash
python -m http.server 8080
```

Open `http://localhost:8080` in your browser.

## QA Mode

Append `?qa=1` to URL to disable scroll animations and show all content immediately.
